카메라 **노출 3요소**(ISO, 셔터 속도, 조리개)는 사진 촬영의 핵심입니다. 이 세 가지를 이해하고 활용하면 원하는 밝기와 효과를 가진 사진을 찍을 수 있어요. 아래에 각 요소를 **쉽게 이해할 수 있는 이미지와 함께** 자세히 설명드릴게요.

![카메라 노출 3요소](/assets/img/posts/exposure-triangle/srl.jpeg)

## 조리개(Aperture)
- 역할 : 렌즈가 빛을 받아들이는 구멍의 크기를 조절하고, 피사체의 선명도(심도)를 결정합니다.
- 표기 : f/값 (예: f/1.8, f/5.6)
- 비유 : 조리개는 창문 크기와 비슷해서 창문이 크면 빛이 많이 들어오고, 창문이 작으면 빛이 적게 들어옵니다.
- 특징 :
  - 낮은 F값 (예: f/1.8):
    - 조리개 구멍의 크기가 커지며, 빛이 많이 들어옴 → 어두운 환경에서 유리.
    - 얕은 심도 → 피사체는 선명하고 배경은 흐림(인물 사진에 적합).
  - 높은 F값 (예: f/11, f/16):
    - 조리개 구멍의 크기가 작아지며, 빛이 적게 들어옴 → 밝은 환경에서 사용.
    - 깊은 심도 → 피사체와 배경 모두 선명(풍경 사진에 적합).
  - 팁 :
    - 인물 사진에서는 f/1.8~f/2.8로 배경 흐림 효과를 강조.
    - 풍경 사진에서는 f/8~f/16으로 선명한 배경.

## 셔터 속도(Shutter Speed)
- 역할 : 셔터가 열리고 닫히는 시간을 조절하여 빛의 양과 움직임을 표현합니다.
- 표기 : 초 단위 (예: 1/100, 1/10, 1초)
- 비유 : 셔터 속도를 눈 깜빡임에 비유할 수 있어요. 눈을 빨리 감으면 순간의 장면만 보이고, 천천히 감으면 움직임이 흐릿하게 보입니다.
- 특징 :
  - 빠른 셔터 속도 (예: 1/1000초) : 빠르게 움직이는 물체(스포츠, 새)를 선명하게 포착.
  - 느린 셔터 속도 (예: 1초 이상) : 움직임을 길게 표현(야경에서 자동차 불빛 궤적, 폭포의 부드러운 물 흐름).
- 팁 :
  - 삼각대가 없으면 셔터 속도를 너무 느리게 설정하지 마세요. 손 떨림으로 인해 사진이 흔들릴 수 있습니다.
  - 움직이는 피사체는 빠른 셔터 속도로 선명하게, 풍경 사진은 느린 셔터 속도로 분위기를 표현하세요.
  - 야간에는 빛이 적기 때문에 셔터 속도를 느리게 설정하세요.

## ISO(감도)
- 역할 : 센서가 빛을 얼마나 민감하게 받아들이는지를 결정합니다.
- 표기 : 숫자 값 (예: ISO 100, ISO 1600)
- 비유 : ISO를 귀에 비유하면, 낮은 ISO는 조용한 환경에서만 잘 들리고, 높은 ISO는 시끄러운 환경에서도 작은 소리를 잘 들으려는 상태입니다.
- 특징 :
  - 낮은 ISO (예: ISO 100, 200): 밝은 낮, 노이즈가 적고 깨끗한 이미지.
  - 높은 ISO (예: ISO 1600, 3200): 어두운 곳에서 사용, 하지만 사진에 노이즈가 생길 수 있음.
- 팁 :
  - 밝은 환경에서는 ISO를 최대한 낮게 설정하세요.
  - 어두운 환경에서는 노이즈를 감수하고 ISO를 올리되, 삼각대 사용으로 낮은 ISO를 유지하면 좋아요.

## 노출 삼각형
이 세 가지 요소는 서로 밀접하게 연결되어 있으며, 적절한 밸런스를 맞추는 것이 중요한 사진의 기본입니다. **“노출 삼각형(Exposure Triangle)”** 이라는 개념을 통해 이들의 관계를 이해할 수 있습니다.
- 예시 :
  - 밝은 환경: ISO를 낮추고, 셔터 속도를 빠르게 하거나 조리개를 조여 빛을 줄입니다.
  - 어두운 환경: ISO를 높이거나, 셔터 속도를 느리게 하거나, 조리개를 열어 빛을 더 받습니다.
  - 이 세 가지 요소로 인해 동일한 노출로 맞추더라도 사진의 느낌을 다르게 표현할 수 있습니다.

## 촬영 시 참고 예시
- 실외 풍경 사진 :
  - ISO 100, 셔터 속도 1/125초, 조리개 f/11
  - 밝고 선명한 사진 촬영.
- 인물 사진 :
  - ISO 200, 셔터 속도 1/250초, 조리개 f/2.8
  - 배경은 흐릿하고 피사체는 뚜렷.
- 야경:
  - ISO 400, 셔터 속도 5초, 조리개 f/8
  - 빛의 궤적을 길게 표현.
